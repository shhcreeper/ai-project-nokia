## About

Fast Gay Frog Adventure is a retro 3D platformer inspired by the Apple ][ era, featuring:
- Rainbow-colored gay pride themed frog character
- Retro pixel art aesthetic with green monochrome UI
- Fast-paced platforming action
- Collectible rainbow items
- CRT screen effects and scanlines

## How to Play

### Controls

- **WASD** or **Arrow Keys** - Move the frog (forward/backward/left/right)
- **SPACE** - Jump
- **R** - Restart the game

### Objective

1. Navigate through the platforms without falling
2. Collect all the rainbow collectibles (flies/pride items)
3. Reach the final platform (purple/magenta colored) to win
4. Try to get the highest score possible!

### Tips

- The frog is FAST - use quick movements to navigate
- Time your jumps carefully - the physics are realistic
- Collect items for bonus points
- Don't fall off the platforms or you'll respawn at the start
- The camera follows you - focus on the frog's movement

## Running the Game

### Local Setup

1. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, Edge)
2. Click "START GAME" to begin
3. Use keyboard controls to play

### Online Hosting

To host online:
1. Upload the entire `frog-game` folder to any web server
2. Ensure all files (index.html, style.css, game.js) are in the same directory
3. Access via the web URL

**Note:** The game uses Three.js via CDN, so an internet connection is required for the 3D graphics library.

## Technical Details

### Built With

- **Three.js** (r128) - 3D graphics rendering
- **Vanilla JavaScript** - Game logic
- **HTML5 Canvas** - Rendering surface
- **CSS3** - Retro UI styling

### Features

- 3D WebGL rendering with pixelated aesthetic
- Physics-based movement and jumping
- Collision detection for platforms and collectibles
- Dynamic camera following the player
- Retro green monochrome UI inspired by Apple ][
- CRT screen effects with scanlines
- Rainbow gradient animations
- Responsive design

### Browser Compatibility

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

**WebGL required** - Make sure hardware acceleration is enabled in your browser.
