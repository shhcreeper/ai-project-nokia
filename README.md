# Wonderlands√óShowtime Rhythm Game

A Project Sekai-themed rhythm game featuring Rui and Tsukasa with falling notes gameplay.

## How to Play

1. Open `index.html` in a modern web browser (Chrome, Firefox, Edge, Safari)
2. Click "Play" to start
3. Select a song and difficulty
4. Press D, F, J, K keys to hit the falling notes when they reach the target zones
5. Time your keypresses for Perfect, Great, or Good judgments

## Features

- **4-Lane Rhythm Gameplay**: Press D, F, J, K keys to match falling notes
- **3 Songs**: Star of the Show, Inventor's Melody, Wonderful Showtime
- **3 Difficulty Levels**: Easy, Normal, Hard
- **Scoring System**: Perfect (300pts), Great (200pts), Good (100pts)
- **Combo System**: Build combos for bonus points
- **Story Mode**: Dialogue scenes with Rui and Tsukasa
- **Settings**: Adjust note speed, music volume, and SFX volume
- **Results Screen**: View your score, accuracy, and grade

## Game Controls

- **D** - Hit notes in lane 1 (leftmost)
- **F** - Hit notes in lane 2
- **J** - Hit notes in lane 3
- **K** - Hit notes in lane 4 (rightmost)

## Timing Windows

- **Perfect**: ¬±50 pixels - 300 points
- **Great**: ¬±100 pixels - 200 points
- **Good**: ¬±150 pixels - 100 points
- **Miss**: Beyond good range - 0 points, combo resets

## Technical Details

- Built with HTML5, CSS3, and JavaScript
- Canvas-based rendering for smooth animations
- Web Audio API for sound effects
- LocalStorage for high score persistence
- No external dependencies required

- Firefox 88+
- Edge 90+
- Safari 14+

## Installation

No installation required! Simply:
1. Extract all files to a folder
2. Open `index.html` in your browser
3. Start playing!
